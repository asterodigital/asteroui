# AsteroUI

**AsteroUI** is a complete Bootstrap-based front and backend template system — purpose-built for community platforms, homeowner associations, and modern web applications.

> The full source code is currently under active development and will be released soon. In the meantime, pre-built distribution assets are available via [jsDelivr CDN](https://cdn.jsdelivr.net/npm/asteroui@1.0.2).

---

## CDN

All distribution assets are published to npm and served via jsDelivr:

```
https://cdn.jsdelivr.net/npm/asteroui@1.0.2
```

### CSS

**Full**

```html
<!-- Unminified -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/asteroui@1.0.2/dist/styles/asteroui.css">

<!-- Minified -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/asteroui@1.0.2/dist/styles/asteroui.min.css">

<!-- RTL -->
```

## Available Variants

| Variant | Description |
|---------|-------------|
| `asteroui` | Full build with all components and styles |
| `asteroui-lite` | Lightweight build for minimal footprint |

Each variant ships with standard, minified, and RTL-ready CSS, plus UMD and ESM JavaScript bundles.
